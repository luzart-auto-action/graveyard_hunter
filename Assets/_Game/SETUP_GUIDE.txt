==============================================================================
                    GRAVEYARD HUNTER - SETUP GUIDE
                  Stealth / Survival / Maze Escape
==============================================================================

TONG QUAN
---------
Game: Graveyard Hunter
The loai: Stealth / Survival / Maze Escape
Goc nhin: Top-down 3D
Phong cach: Low-poly, tong mau toi, anh sang noi bat

YEAU CAU
--------
- Unity 2021.3+ (LTS khuyen nghi)
- DOTween (da co san trong project)
- Odin Inspector (da co san trong project)
- TextMeshPro (Unity built-in)
- NavMesh Components (Unity AI Navigation package)

==============================================================================

Tat ca scripts da duoc tao tai: Assets/_Game/Scripts/
Folder structure:

Assets/_Game/
├── Art/Materials/          <- Tao boi Material Factory
├── Prefabs/               <- Tao boi Prefab Creator
│   ├── Characters/
│   ├── Environment/
│   ├── Traps/
│   ├── Boosters/
│   ├── FX/
│   └── UI/
├── Scenes/
├── ScriptableObjects/
│   ├── Configs/           <- GameConfig.asset
│   ├── Levels/            <- Level_01 -> Level_10
│   └── Skins/
└── Scripts/
    ├── Core/              <- GameManager, EventBus, ServiceLocator, ObjectPool
    ├── GameState/         <- GameStateManager + 7 states
    ├── Level/             <- LevelManager, MazeGenerator, ExitGate, TreasurePickup
    ├── Player/            <- PlayerController, PlayerHealth, PlayerInventory, PlayerLightSystem
    ├── Enemy/             <- LightGhost, GhostLightCone, GhostEyes
    ├── Input/             <- InputManager, VirtualJoystick
    ├── UI/                <- UIPanel, UIManager, 7 panels
    ├── Data/              <- GameConfig, PlayerProgressData, SkinData
    ├── Audio/             <- AudioManager
    ├── CameraSystem/      <- CameraController

    ├── Command/           <- ICommand, CommandManager
    ├── Trap/              <- TrapBase + 3 trap types
    ├── Booster/           <- BoosterBase + 4 booster types
    ├── Shop/              <- ShopManager
    └── Editor/            <- 5 editor tools

==============================================================================
                   BUOC 7: SETUP NAVMESH
==============================================================================

Quan trong cho Enemy AI (Ghost dung NavMeshAgent):

1. Mo Window > AI > Navigation
2. Chon tat ca Wall prefabs va Floor:
   - Wall: danh dau Navigation Static, Not Walkable
   - Floor: danh dau Navigation Static, Walkable
3. Tab Bake:
   - Agent Radius: 0.5
   - Agent Height: 2
   - Step Height: 0.4
   - Click "Bake"
4. NavMesh phai duoc bake TRUOC khi play

LUU Y: Moi lan LoadLevel se tao maze moi, can bake NavMesh dynamically.
- Su dung NavMeshSurface component (Unity AI Navigation package)
- Them NavMeshSurface vao Floor parent object
- Goi NavMeshSurface.BuildNavMesh() sau khi tao maze

                    ARCHITECTURE NOTES
==============================================================================

PATTERNS SU DUNG:
- Service Locator: Managers dang ky/truy cap qua ServiceLocator
- EventBus (static pub/sub): Giao tiep giua cac he thong
- State Machine: GameStateManager quan ly GameState
- Factory Pattern: Prefabs qua PrefabCreator
- Object Pool: FX, AudioSources
- ScriptableObject: GameConfig, LevelData, SkinData

QUY TAC VANG:
- GameManager la orchestrator DUY NHAT subscribe user-action events
- Workers (LevelManager, etc.) KHONG subscribe events, chi expose public methods
- GameplayUI dung CanvasGroup (KHONG SetActive)
- WinPanel subscribe trong Awake() va cache data
- Buttons trong Win/Fail KHONG goi Hide()
- ForceHideAllPanels() truoc moi transition lon
- DestroyImmediate() khi clear level
- DOTween.KillAll(false) TRUOC destroy objects
- .SetUpdate(true) cho TAT CA UI animations

==============================================================================
                    TROUBLESHOOTING
==============================================================================

LOI THUONG GAP:

1. "Service not found" error:
   - Dam bao tat ca Managers da co trong Scene
   - Kiem tra thu tu Awake: ServiceLocator.Register phai chay truoc Get
   - Dung Script Execution Order: ObjectPool (-100), Managers (-50), GameManager (0)

2. Ghost khong di chuyen:
   - Kiem tra NavMesh da bake chua
   - Them NavMeshSurface component va goi BuildNavMesh()
   - Kiem tra NavMeshAgent component tren Ghost prefab

3. UI khong hien thi:
   - Kiem tra Canvas co CanvasScaler khong
   - Kiem tra EventSystem ton tai
   - Kiem tra panel _panelName dung ten

4. Player khong tuong tac voi Treasure/ExitGate:
   - Kiem tra Tags: "Player", "Treasure", "ExitGate"
   - Kiem tra Collider la Trigger
   - Kiem tra Rigidbody (it nhat 1 object can co)

5. DOTween errors khi chuyen level:
   - Dam bao goi DOTween.KillAll(false) trong ClearLevel()
   - Dung DestroyImmediate thay vi Destroy

6. UI Panel animations khong chay khi Pause:
   - Kiem tra .SetUpdate(true) tren tat ca UI tweens
